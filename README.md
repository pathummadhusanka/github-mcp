**About this repository**

Repository created using GitHub MCP and GitHub Copilot.

This repo demonstrates a minimal Python "Hello, world!" app that was scaffolded and tested using automation and AI-assisted tools. The `hello.py` program and its test were added, then an issue and pull request were created and merged using GitHub MCP and Copilot during the workflow.

Key points:

- Created with GitHub MCP and Copilot assistance.
- Issue and PR were opened and merged as part of the demo workflow.
- Includes `hello.py` and a simple `pytest` test.
